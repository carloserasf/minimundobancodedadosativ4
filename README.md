# 🍗 Projeto SQL - American Chicken Delivery

Este repositório contém todos os scripts SQL criados para o exercício de **Criação e Manipulação de Dados (SQL)** baseado no minimundo do delivery American Chicken.

---

## 📁 Estrutura do Repositório

american-chicken-sql/
│
├── create_database.sql → Criação do banco e das tabelas
├── insert_data.sql → Inserção de dados iniciais
├── select_queries.sql → Consultas SELECT (JOIN, WHERE, ORDER BY)
├── update_delete.sql → Comandos UPDATE e DELETE
└── README.md → Documentação do projeto


---

## 🚀 Como executar

1. Abra o MySQL Workbench ou PGAdmin.
2. Execute primeiro o arquivo **create_database.sql**
3. Depois execute:
   - `insert_data.sql`
   - `select_queries.sql`
   - `update_delete.sql`
4. Tudo será criado automaticamente.
